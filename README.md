## Sustc-classes-rating
一个学校课程评分网站，打算复习过去一年所学，同时点亮新的技能树

### 计划使用的技术栈 
网站：Vue + lumen + MySQL
数据抓取: Scrapy

### 复习内容
- Docker
- Vue的基本用法
- PHP基础
- 数据库操作
- Scrapy基本操作

### 可能涉及的新内容
- TypeScript
- Vue深入
- 成熟框架使用（lumen)

### 开发过程中涉及的坑、新技能、工具
- nginx工作目录为src，但访问根目录设置到src/public
- lumen作为lavarel精简版部分功能的缺失
- Vue与lumen的整合
- lavarel-elixir
